Biruni

A free, native macOS file manager inspired by the classic dual-pane DOS file managers of the 1980s — fast, keyboard-driven, and unapologetically retro.

Download for macOS View source on GitHub
Free · macOS 12 Monterey or later · Apple Silicon & Intel

Why Biruni

Dual-pane, keyboard-first

Two side-by-side panels and full F1–F10 key coverage — copy, move, rename, delete, and view files without reaching for the mouse.

Built-in editor & viewer

F3 opens a quick viewer (text or hex+ASCII for binaries); F4 opens a real inline editor right in the opposite panel.

Real command line

A shell prompt at the bottom runs through your actual login shell — Homebrew, nvm, aliases, all of it — right in the active panel's folder.

Archive browsing

Step into .zip, .tar, .tar.gz, and .tgz files like a folder, and extract marked items straight to the other panel.

Compare directories

Mark everything that differs — or is missing — between the two panels in one keystroke.

Classic DOS palette

Blue background, white directory brackets, magenta archives, yellow marks, cyan cursor bar — faithful to the original.

Get Biruni

Download the latest release

Unzip it and drag Biruni.app into your Applications folder.

First launch

Biruni isn't notarized yet, so macOS will show a "Biruni' Not Opened" warning the first time you open it — that's expected for a small independent app distributed outside the App Store. To open it:

Right-click (or Control-click) Biruni.app in Finder → choose Open → confirm in the dialog that appears. You only need to do this once.

Biruni is named for Abu Rayhan al-Biruni (973–1048), a Persian polymath from Khwarazm. Source on GitHub
